Как это работает

KREEZ — это графическая оболочка над winws.exe (проект zapret).
Приложение автоматически скачивает нужные бинарники, управляет процессом обхода
и позволяет настраивать стратегии без терминала.

⟳
АВТО-ЗАГРУЗКА
При первом запуске KREEZ сам скачивает zapret с GitHub. Ничего не нужно устанавливать вручную.
◎
DPI-ОБХОД
Использует WinDivert для перехвата пакетов и обмана Deep Packet Inspection провайдера.
⊞
БЕЗ VPN
Работает на уровне сетевых пакетов. Не замедляет соединение, не требует сторонних серверов.
✦
СВОИ СЕРВИСЫ
Добавляй любые домены и IP-диапазоны через встроенный редактор. Полный контроль над списками.

Первый запуск

Три шага от установки до работающего обхода.

01
Запусти KreezApp.exe от имени администратора
Правой кнопкой → «Запустить от имени администратора».
WinDivert требует права администратора для перехвата пакетов.
Если запустить без прав — обход не запустится, появится ошибка.
02
Дождись автозагрузки zapret
При первом запуске KREEZ автоматически скачает winws.exe, WinDivert.dll и драйвер WinDivert64.sys с GitHub.
Прогресс отображается в главном окне. Обычно занимает 10–30 секунд.
03
Нажми [ START ]
Включи нужные сервисы во вкладке ⊞ Сервисы, выбери стратегию в ◉ Стратегия и нажми [ START ] на главной вкладке.
Зелёная точка и статус Обход активен — всё работает.

Что где находится

Нажми на раздел чтобы развернуть описание.

⌂ ГЛАВНАЯ ▶

Центр управления обходом. Отсюда запускаешь и останавливаешь winws.exe.

  • Статус — цветная точка: серая (стоп), оранжевая (запуск), зелёная (активен), красная (ошибка)
  • [ START ] — запустить обход с текущей стратегией и включёнными сервисами
  • [ STOP ] — остановить winws.exe
  • [ RESTART ] — перезапуск (применяет новые настройки без перехода в другую вкладку)
  • [ check updates ] — проверить обновления zapret на GitHub
  • Лог-консоль — вывод winws.exe в реальном времени, полезно для диагностики
⊞ СЕРВИСЫ ▶

Список встроенных сервисов. Включай только те, которые нужно обойти — лишние нагружают winws впустую.

  • YouTube — youtube.com, googlevideo.com и CDN
  • Discord — discord.com, CDN, голосовые серверы
  • Telegram — t.me и telegram.org
  • Faceit — серверы EU (185.25.204.0/24)
  • Rust — игровые серверы
  • Amazon / Prime Video — primevideo.com, cloudfront.net
  • Twitch — twitch.tv и CDN
  • OpenAI / ChatGPT — openai.com, api.openai.com

Переключатель справа — toggle вкл/выкл. Изменения применяются после перезапуска обхода.

◈ IP-АДРЕСА ▶

Ручное управление IP-листом (ipset). Используется для точечного обхода по IP-диапазонам когда домена недостаточно.

  • Вводи адреса в формате 1.2.3.4 или 1.2.3.0/24
  • Описание — опционально, для твоих пометок
  • Кнопки [ IMPORT ] и [ EXPORT ] — загрузка/выгрузка списка в .txt
  • Кнопка ✕ справа — удалить конкретный адрес
◉ СТРАТЕГИЯ ОБХОДА ▶

Метод, которым winws.exe обманывает DPI провайдера. Разные провайдеры реагируют на разные стратегии — попробуй несколько если одна не работает.

  • AUTO — комбинация fake + multisplit, подходит большинству. Начни с него.
  • FAKE TLS — подменяет TLS ClientHello фейковым пакетом с TTL=5
  • MULTISPLIT — разбивает первый пакет на фрагменты, DPI не успевает собрать
  • DISORDER — disorder2 + multisplit, более агрессивный вариант
  • FAKE QUIC — обход UDP/QUIC блокировок (YouTube нередко использует QUIC)
  • COMBO MAX — три блока: HTTPS + HTTP + Discord QUIC одновременно

Секция Ручные аргументы — для продвинутых пользователей. Включи toggle и введи аргументы напрямую в winws.exe. Полностью заменяют выбранную стратегию.

--wf-tcp=80,443 --dpi-desync=fake,multisplit --dpi-desync-ttl=5
✦ СВОИ СЕРВИСЫ ▶

Добавляй любые сайты и серверы которых нет в стандартном списке. KREEZ сгенерирует нужные файлы списков автоматически.

  • Название — любое, для тебя
  • Цвет — HEX-код, цветная полоска в списке
  • Описание — заметка, необязательно
  • Домены — по одному на строку, например example.com
  • IP-диапазоны — по одному на строку, например 1.2.3.0/24

После добавления KREEZ запишет домены в list-custom-domains.txt и IP-адреса в ipset-custom-services.txt. Перезапусти обход чтобы применить.

Кнопка ✎ — редактировать существующий сервис. Кнопка ✕ — удалить.

⚙ НАСТРОЙКИ ▶

Системные настройки приложения.

  • Автозапуск — запускать KREEZ при старте Windows (через реестр HKCU)
  • Авто-обновление — автоматически проверять обновления zapret при запуске
  • Интервал обновлений — как часто проверять (ежедневно / еженедельно / ежемесячно)
  • Путь к zapret — путь к папке bin с winws.exe. Обычно не нужно менять вручную.
  • Тема — системная / светлая / тёмная

Нажми [ SAVE ] чтобы сохранить изменения.

Какую стратегию выбрать

Начни с AUTO — он работает у большинства российских провайдеров. Если не помогает — пробуй по порядку сверху вниз.

AUTO РЕКОМЕНДУЕТСЯ
Fake TLS + multisplit комбо. Оптимальный баланс совместимости и скорости. Работает у большинства провайдеров.
FAKE TLS
Подменяет TLS-рукопожатие. Эффективен когда провайдер анализирует только первый пакет соединения.
MULTISPLIT
Фрагментирует первый пакет. DPI не успевает собрать данные для анализа. Хорошо работает на высоких скоростях.
DISORDER
Disorder2 + multisplit. Более агрессивный вариант — отправляет пакеты в неправильном порядке с намеренным перекрытием.
FAKE QUIC
Только для UDP/QUIC (порт 443 UDP). YouTube и Google активно используют QUIC — этот режим помогает когда TCP-стратегии не справляются.
COMBO MAX МОЩНЫЙ
Три блока одновременно: HTTPS (TCP 443), HTTP (TCP 80) и Discord QUIC (UDP 443). Максимальное покрытие, чуть выше нагрузка на CPU.

Не знаешь что выбрать? Спроси в Discord — скажи своего провайдера и что блокируется.

Частые вопросы

Антивирус ругается на WinDivert — это опасно? +
Нет. WinDivert — это легитимный Windows-драйвер для перехвата сетевых пакетов. Антивирусы срабатывают из-за эвристики (драйвер ядра + перехват пакетов = подозрительно). Добавь папку KREEZ в исключения антивируса. Если хочешь убедиться — исходный код winws открыт на GitHub (zapret by bol-van).
Обход не запускается, пишет «процесс завершился сразу» +
Самые частые причины:
1. Запущено без прав администратора — перезапусти от имени администратора.
2. Антивирус заблокировал WinDivert64.sys — добавь в исключения.
3. Другой экземпляр winws.exe уже работает — открой Диспетчер задач и завершите его.
4. Путь к zapret указан неверно — зайди в Настройки и исправь.
YouTube работает, но Discord нет (или наоборот) +
Убедись что нужный сервис включён во вкладке ⊞ Сервисы. Попробуй разные стратегии — Discord часто требует другой подход чем YouTube. COMBO MAX включает одновременно несколько методов для разных протоколов. Если ничего не помогает — напиши в Discord с указанием провайдера.
Замедляет ли KREEZ интернет? +
Незначительно. winws.exe обрабатывает только первые несколько пакетов соединения, основной трафик проходит без обработки. На обычном ПК нагрузка на CPU составляет 0–2%. На очень медленных процессорах или при большом количестве одновременных соединений возможно небольшое снижение скорости.
Как добавить свой сайт в обход? +
Вкладка ✦ Свои сервисы → введи название → добавь домены (по одному на строку) и/или IP-диапазоны → нажми [ добавить сервис ]. После этого нажми [ RESTART ] на главной вкладке чтобы применить изменения.
Где хранятся настройки? +
Все настройки хранятся в файле config.json рядом с KreezApp.exe. Zapret (winws.exe и списки) скачивается в папку zapret/ там же. Кастомные списки — в zapret/lists/list-custom-domains.txt и zapret/lists/ipset-custom-services.txt.
Работает ли с корпоративной сетью / VPN? +
В корпоративных сетях провайдером является корпоративный файрвол, а не домашний ISP. Результаты варьируются. С VPN обход обычно не нужен — VPN уже шифрует трафик. Запускать KREEZ одновременно с VPN не рекомендуется.
